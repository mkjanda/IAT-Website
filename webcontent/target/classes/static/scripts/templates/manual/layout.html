The screens to the right, which is accessible via the Layout option under the View drop-down menu, allows you to modify the size of your IAT testbox within the webpage as well as allowing you to modify the size of elements within the test. Simply click on any of the dropdown boxes to change the values. Values that appear in gray will result in overlap of test elements.

In the event of overlap, elements in the test will appear in red as they do in the lower picture and you will have to correct this in order to save your settings.

You might find this feature of resizing your test elements particularly useful if you wish to include images in your response keys, an example of which shown on the Response Keys section.
